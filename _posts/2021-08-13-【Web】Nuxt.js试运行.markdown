## 新建nuxt项目
使用`create—nuxt-app`来建立nuxt项目。
`mkdir nuxt`
npx create-nuxt-app --overwrite-dir .
Since I can't create the nuxt project in a non-empty directory
`npx create-nuxt-app .`
